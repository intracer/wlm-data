Ukraine: Populated Places
Source: https://data.humdata.org/dataset/ukraine-populated-places
Populated places (settlements) with coordinates and types are derived from Ukraine COD-AB dataset shared by SSPE “Kartographia”

ADM_PCODEs are given according to KATOTTH (https://en.wikipedia.org/wiki/Codifier_of_administrative-territorial_units_and_territories_of_territorial_communities)

This data is used to get an English name by KATOTTH code.
